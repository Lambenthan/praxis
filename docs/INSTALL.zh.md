# Praxis 安装指南

写给第一次用 AI 工具的社科研究者。全程不需要命令行,预计 10 分钟。

## 你需要准备什么

- 一台 Apple Silicon 的 Mac(M1/M2/M3/M4)。
- 一个模型服务商的 API key(下面第 3 步教你申请,DeepSeek 充 10 块钱够用很久)。
- 如果要跑 Stata 分析:这台电脑上已经装好、能正常打开的 Stata(MP/SE/BE 都行)。没有 Stata 也能用,R 和 Python 分析不需要它。

## 第 1 步:安装

1. 下载 `Praxis_x.x.x_aarch64.dmg`,双击打开。
2. 把 Praxis 图标拖进 Applications 文件夹。
3. **第一次打开要多一步**:在"应用程序"里**右键点击 Praxis → 选"打开" → 再点"打开"**。
   直接双击会被 macOS 拦下来提示"无法验证开发者",这是因为应用没有付费的苹果签名,不是病毒。右键打开只需要做这一次,以后正常双击即可。
   如果右键也打不开:系统设置 → 隐私与安全性 → 拉到最下面,点"仍要打开"。

## 第 2 步:跟着向导走

第一次启动会自动进入"安装向导"页(以后也能从左下角"安装向导"再进来)。向导只有三步:

### ① 连接一个模型

助手背后需要一个 AI 模型,就像手机需要装一张电话卡。

1. 选一家服务商。不知道选哪家就选 **DeepSeek**:便宜、稳定、国内直连。
2. 点"去申请 key",在打开的网页上注册账号 → 充值一点余额 → 创建一个 API key(一串 `sk-` 开头的字符)。
3. 把这串字符粘贴回向导,点保存。显示"已连接"就成了,默认模型会自动帮你选好。

### ② 启用 Stata(可选)

电脑里装了 Stata 就点一下"启用 Stata"。它会把 Stata 桥接装进一个独立环境(第一次要下载一个独立的 Python,等一两分钟),装完助手就能替你写 do 文件、跑回归、读日志。

向导会告诉你有没有检测到本机的 Stata。没装 Stata 就跳过这步。

### ③ 跑一遍演示

点"打开演示",消息已经替你填好,按发送。它会用 Stata 自带的 auto 数据跑三个回归模型,一两分钟后结果表格会自动出现在窗口右侧——你在表格里逐列点"采纳"或"拒绝",定下终稿,再一键导出 LaTeX / Word / do 文件。

看到那张表,整条链路就通了。

## 之后怎么用

- 左上角"新建"开新对话。欢迎页上有现成的入口:研究设计、分析你的数据、基准回归、质性编码,也可以直接用自己的话提需求。
- 每个对话在你的"工作区文件夹"里有自己的日期子文件夹,数据、do 文件、结果都落在里面,随时可查可复现。
- 更多能力在"设置"里:换模型、接入 Zotero / 文献检索 / FRED 数据、一键启用 Jupyter(跑 R / Python 笔记本)。

## 常见问题

**打开后一直显示"正在启动"?**
等 10 秒左右;还不行就退出重开一次。

**发消息报模型错误?**
多半是 key 没余额或粘贴时带了空格。去服务商网站确认余额,回"设置 → Model"重新粘贴。

**点"启用 Stata"失败?**
确认 Stata 本体能正常打开;第一次启用需要联网下载,网络不稳时重试一次即可。

**我的数据会被传到哪里?**
文件都在你自己的电脑上。只有发给助手的消息和它需要读的内容会发送给你选的模型服务商(和你直接用该家的网页版一样),设置页的"Privacy & data flow"里写明了每一项数据的去向。
